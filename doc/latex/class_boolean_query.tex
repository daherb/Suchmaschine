\subsection{Detailed Description}
The \hyperlink{class_boolean_query}{Boolean\-Query} class is the Implementation of a basic \hyperlink{class_query}{Query$<$int$>$} Parser. A \hyperlink{class_boolean_query}{Boolean\-Query} String consists of Document Terms connected by the boolean Operators A\-N\-D, O\-R and N\-O\-T. The \hyperlink{class_query}{Query} String is evaluated from left to right.
