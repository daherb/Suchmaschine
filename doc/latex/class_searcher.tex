\subsection{Detailed Description}
The \hyperlink{class_searcher}{Searcher} class is the basic Class of the Search Engine. It provides Interface to and keeps track of the Search \hyperlink{class_index}{Index}, the \hyperlink{class_query}{Query}, etc.
