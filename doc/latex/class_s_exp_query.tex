\subsection{Detailed Description}
The \hyperlink{class_s_exp_query}{S\-Exp\-Query} class is the Implementation of a basic \hyperlink{class_query}{Query$<$int$>$} Parser. A \hyperlink{class_s_exp_query}{S\-Exp\-Query} String is quite similar to Lisp-\/like expressions. The form of a \hyperlink{class_query}{Query} Expression is (o e1 e2) with o one of A\-N\-D, N\-O\-T, O\-R and e1,e2 either Document Terms or \hyperlink{class_query}{Query} Expressions.
